🛠 План розробки проєкту: [Консольна гра "Змійка"]

1. Концепція продукту
  Тип додатку: Десктоп (Консольна гра)
  Основна ідея: Клон класичної гри "Змійка" в текстовому режимі. Гравець керує змійкою за допомогою клавіш WASD, збирає яблука на полі, змійка    росте. Гра закінчується при зіткненні зі стіною або з власним тілом. Мета — з'їсти якомога більше яблук.
2. Основний функціонал (Features)
  Відображати ігрове поле зі стінами по краях (15×10 символів)
  Змійка рухається автоматично в обраному напрямку
  Керування клавішами W (вгору), S (вниз), A (вліво), D (вправо) без натискання Enter
  На полі випадковим чином з'являється яблуко
  При поїданні яблука змійка збільшується на 1 клітинку
  Лічильник очок (score) збільшується на 1 за кожне яблуко
  Перевірка зіткнень: змійка vs стіни, змійка vs власне тіло
  Екран завершення гри з фінальним рахунком
  Можливість вийти з гри клавішею Q
  Крос-платформність (працює на Windows, Mac, Linux)
